# 📱 외부 기기(스마트폰/태블릿) 접속 가이드

유나의 식단 관리 앱을 스마트폰에서도 편리하게 사용해보세요!

## 1. 접속 방법
1. **서버 실행**: PC에서 `app.py` 또는 `main_desktop.py`를 실행합니다.
2. **Wi-Fi 확인**: 서버 PC와 스마트폰이 **동일한 Wi-Fi 공유기**에 연결되어 있는지 확인합니다.
3. **주소 입력**: 스마트폰 브라우저(Chrome, Safari 등) 주소창에 터미널에 표시된 모바일 접속 주소를 입력합니다.
   - 예: `http://192.168.0.10:5000`

## 2. 접속이 안 될 때 (트러블슈팅)

동일한 Wi-Fi인데도 스마트폰에서 접속이 안 된다면 아래 3가지를 확인해주세요.

### ① 윈도우 방화벽에 'Python' 프로그램 등록
포트만 열어서 안 되는 경우, 서버를 실행하는 **Python 프로그램 자체**가 차단되어 있을 수 있습니다.
1. [제어판] -> [시스템 및 보안] -> [Windows 방화벽에서 앱 허용] 클릭.
2. [설정 변경] 버튼 클릭 -> [다른 앱 허용] 클릭.
3. `python.exe` 파일 경로를 찾아 추가해야 합니다. (보통 파이썬이 설치된 경로)
   - 찾기 어려우시면 PowerShell(관리자)에서 아래 명령어를 실행해 보세요:
     ```powershell
     netsh advfirewall firewall add rule name="Python_Flask" dir=in action=allow program="C:\Users\new\AppData\Local\Programs\Python\Python312\python.exe" enable=yes
     ```
     *(※ 경로는 사용자 PC의 실제 Python 설치 위치에 따라 다를 수 있습니다.)*

### ② 아이폰/안드로이드 설정 확인 (중요!)
최신 스마트폰은 보안을 위해 **'로컬 네트워크 접근'**을 명시적으로 허용해야 할 때가 있습니다.
- **아이폰:** [설정] -> [개인정보 보호 및 보안] -> [로컬 네트워크] -> 사파리나 사용 중인 브라우저가 켜져 있는지 확인.
- **VPN 확인:** 핸드폰에 VPN이나 'iCloud 비공개 릴레이'가 켜져 있다면 접속이 안 됩니다. 잠시 꺼주세요.

### ③ 타사 백신(V3, 알약 등) 확인
노트북에 V3나 알약 같은 별도의 보안 프로그램이 있다면, **그 프로그램의 방화벽**이 윈도우 설정을 무시하고 막고 있을 확률이 매우 높습니다. 테스트를 위해 잠시만 백신 프로그램을 종료하거나 실시간 감시를 끄고 접속해 보세요.

### ④ 공유기 'AP 격리' 기능
만약 다른 기기끼리 서로 아예 못 보게 하는 기능이 공유기에 켜져 있다면 접속이 안 됩니다. 이는 공유기 관리자 페이지에서 확인이 필요합니다.

이제 주방이나 외출 중에도 유나의 식단을 간편하게 기록하고 확인하세요! 🥣✨
